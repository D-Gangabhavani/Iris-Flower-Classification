# Iris Flower Classification

## Project Overview
This is a Machine Learning project that classifies Iris flowers into three different species: **Setosa, Versicolor, and Virginica**. The classification is done based on the measurements of the flower's sepals and petals.

## Domain
* Machine Learning (Internship Project)

## Features
* Loads and analyzes the Iris flower dataset.
* Trains a Machine Learning model to classify the flowers.
* Provides accurate predictions based on input data.

## Technologies Used
* **Python**: The core programming language.
* **Scikit-Learn**: For building and training the Machine Learning model.
* **Pandas**: For data handling and analysis.
* **Jupyter Notebook / VS Code**: The development environment.
